# Tariga — design prototype (Claude design export)

An interactive design prototype of the next-generation Tariga UI, exported
from Claude's design tool: mobile-first, voice-first coach with a listening
orb, home dashboard with streak + progress ring, bottom tab navigation, a
"then → now" journey screen, and a word-origins map of Sudan.

**View it:** open `design/Tariga.dc.html` on the deployed site (or locally —
it loads React from unpkg, so it needs internet).

**Status:** this is the visual/interaction reference, not the product — the
coach feedback in here is canned demo data. The working app (repo root) has
the real AI coaching; the plan is to port this design onto it.
